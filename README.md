# Vanguard A/B Test

Analysis of Vanguard's digital interface A/B test (March–June 2017), evaluating whether a redesigned UI improved process completion rates.

## Key findings
- Completion rate: Control 65.58% → Test 69.29% (+3.71 pp)
- Statistically significant (two-proportion z-test, p < 0.001)
- Does not meet Vanguard's 5 pp cost-effectiveness threshold
- Error rate increased in the Test group
